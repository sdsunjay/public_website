DWPACLEC2

Distributed WPA Cracker Leveraging EC2

This project builds upon the work presented by security research Thomas Roth at BlackHat by providing a practical implementation to cracking WPA/2 passwords with CUDA. It does this by incorporating a dictionary attack that allows an attacker to easily store their word list in a database, which the cracking nodes query. This paper describes the implementation of the attack and discusses countermeasures to help mitigate it. All the source code can be found here.
