Product Review Classification with BERT and PyTorch

Classifying Amazon Product Reviews using BERT and PyTorch.

Project Overview

In this project, we followed this guide and used the bert-large-uncased model, which is a version of BERT (Bidirectional Encoder Representations from Transformers), a state-of-the-art natural language processing model, to classify product reviews as written on Amazon.

We wanted to determine if users had encountered a safety issue with the product, such as it being dangerous for children or pets or if it had the potential to cause harm. We implemented two state-of-the-art models, BERT and ALBERT (A Lite BERT), in Python using the PyTorch library, which is a popular open-source machine learning framework. Ensemble learning is a machine learning technique that combines the predictions of multiple models to make more accurate predictions.

Data

Amazon supplied the dataset of hundreds of product reviews with very few reviews indicating a safety issue. The text provided was already labeled, but we did not have access to the labels for the final test dataset. We used a dataset of customer reviews for a particular product. The dataset consisted of text reviews and corresponding labels indicating whether the review was safe or unsafe.

Preprocessing

Before training the model, we preprocessed the data by changing the text to lowercase, removing punctuation, and tokenizing the reviews using the BERT vocabulary. Since BERT is limited to 500 tokens, we needed to remove any unnecessary words and for long reviews, we simply truncated the review. We also split the dataset into training, validation, and test sets.

Training

We trained the BERT model on the training set using the Adam optimization algorithm and a binary cross-entropy loss function. We fine-tuned the model by adjusting the learning rate and the number of epochs. PyTorch provided a convenient and efficient way to build, train, and deploy the model for text classification.

We then used ensemble learning to combine the predictions of the BERT and ALBERT models. We experimented with different ensemble methods, such as simple averaging and weighted averaging, and selected the method that provided the best performance on the validation set.

Conclusions

We evaluated the model on the test set and obtained an accuracy of 84.25% and f1-score of 77.05% respectively.

The BERT model achieved good performance on the task of classifying product reviews as dangerous or not dangerous. This can be useful for businesses seeking to gather customer feedback and improve their products and services. The PyTorch library made it easy to implement and train the model for this task.
