Picture Changer

Changing Pictures with JavaScript

This project was started in 2014, but completed in 2023! It involved creating an image slider using HTML, CSS, and JavaScript and is a continuation of this projects. The goal was to display a series of images in a slideshow, with a smooth transition between each image.

To create the image slider, we used a combination of HTML, CSS, and JavaScript. The HTML provided the structure for the image slider, while the CSS controlled the styling and layout of the images. The JavaScript was used to handle the transition effects between the images, as well as to control the timing of the slideshow.

To create the transition effects, we used a JavaScript library called jQuery. This was the first time using jQuery for this project, and it proved to be a useful tool for handling the animations and interactions in the image slider.

Overall, this small project was a success, and we were able to create a visually appealing and user-friendly image slider that showcases a series of images in an interactive slideshow.
