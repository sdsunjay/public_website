Classifying Amazon Product Reviews using BERT and PyTorch.

Project Overview

This project was inspired by my desire to have my terminal project as realistic as possible. As any good programmer knows, Vim is essential when writing code. Even if I am mainly using IntelliJ IDE, I will still occasionally use Vim to make small edits (all development for this website was done using Vim). This is why I knew I had to integrate a very good emulator of it. After a good bit of searching, I found the Code Mirror project, which is in active development to create emulators of popular text editors, such as Vim, Emacs, and others. I downloaded the project and made many edits to it before I was finally able to add it to my site. However, recently my development has fallen behind the main branch of the project and a few features that used to work, no longer do (like full-screen mode). I hope to get back to this project soon, but adjusting the CSS after every download is a huge time suck. The save and load features work and soon the file will become associated with a particular user, assuming you create an account. In the meantime, have fun playing around with Vim, almost all of the shortcuts work as expected.
