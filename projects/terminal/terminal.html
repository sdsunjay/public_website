jsterm 2.0

"Command Line" Emulator Project

This is the project that inspired me to create a website. I thought having a command line emulator in a browser was so cool and so far removed from the everyday ordinary boring web, not to mention functional and intuitive (at least for a developer). It is by far the largest project I have hosted on my website. It is still in concurrent development with my User Accounts project because of how tightly connected they are. This project was originally started by Clark Duvall, a software engineering genius and my classmate at Cal Poly. However, I have expanded a lot on the project and have added many new commands and features. The biggest addition being account creation and interaction. I also integrated this project with Vim, the best text editor ever. My ultimate goal with this project is to have it to be practically identical to the real Unix Shell. This means finishing integration with User Accounts, so that when the filesystem (a simple JSON file) is created, it is associated with an individual user. If you want more technical details, view them here. I highly recommend you check out the project for yourself. I recently took a break from working on it, after working on it extensively over the summer and becoming frustrated with the 'mkdir' command (UPDATE: 'mkdir' is functional now!), but I hope to return to it and complete it soon, preferably in collaboration with Clark.
