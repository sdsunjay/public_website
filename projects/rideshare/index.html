Cal Poly Rideshare

An easier way to get a ride

Currently the easiest way to travel to SLO is by car. However many people (mainly students) do not have cars or would prefer to save money and carpool. The easiest way to setup a carpool is through the Cal Poly Rideshare Facebook group, which has over 11,000 members and is still rapidly growing. Unfortunately due to the prolific posts, finding a ride either leaving or arriving in SLO is difficult as a user must scroll through potentially dozens of posts. This project aims to eliminate that problem.
Using the Facebook SDK for PHP, I have been able to capture all the posts from the Facebook group and using Python's NLTK library pull out the important information. This project is still under development, but you can track my progress and check out the existing code. I hope to finish this project soon, so students can save time and easily find people to give or get rides with.
