Chat GPT Alexa Skill

Interactive voice experience using OpenAI's GPT model with Amazon Alexa.
Chat GPT Alexa Skill Description

Project Background

This project features an Alexa skill that leverages OpenAI's Chat API to respond to user inquiries. To ask a question, users simply say "ask," followed by their query. This input triggers a Python Lambda function on AWS, which interacts with the OpenAI API to obtain the answer. Alexa then vocalizes the response and prompts the user if they wish to continue asking questions. User do not need to continue to say "ask" before asking their follow-up questions.

View Live Alexa Skill

Technical Details

For retrieving questions and answers from a specific date, the skill incorporates custom Python code that parses dates for the AMAZON.DATE slot. Overall, this Alexa skill offers a streamlined and effective method for users to access information through OpenAI's GPT-3.5 model.

The infrastructure for this project was developed using AWS CDK in TypeScript and included several components: S3 for storage, Lambda for serverless compute, Cloudwatch for logging and monitoring dashboard, Secrets Manager for managing sensitive data, such as the OpenAI API Key, and the Alexa skill.

Actions & Outcomes

The skill offers several interactive features: it can repeat the last question and answer, incorporate past conversations into the current dialogue, and fetch questions and answers from a specific date. Users can add to their conversation history by saying, "add conversation history," and retrieve previous questions and answers by stating, "What are my questions and answers," followed by a date or time period. Additionally, users can search for their historical questions with prompts like, "What were my questions on [specific date]?" or "What were my questions last month?"

Conclusions

This project has been an enlightening educational experience, significantly enhancing my understanding of Alexa and user's voice interactions. I was pleasantly surprised by the extensive engagement the Alexa skill garnered, with hundreds of users and thousands of questions processed. I only created this skill with myself as a user, but I was pleased to see so many people using it. This level of interaction not only underlines the robustness and reliability of the skill, but also highlights AWS's scalable and dependable infrastructure, which effortlessly handled the increasing load. The experience has greatly contributed to my professional growth, offering valuable insights into user behavior, the scalability of cloud solutions, and the practical applications of Alexa, ChatGPT, and AWS technology.
